## Planning mode

When the user prompts "Planning mode" or "P mode" or "pmode": do not create, edit, or delete any file unless the user prompts "Proceed". Planning mode means the session is for discussing and designing changes to the agent — not implementing them. Always display proposed changes to the user and ask to proceed.

Read `/aos-router.md` and resolve the active instance before running any workflow.
